### 3. 添加高级 UI 功能

#### 📥 拖拽文件支持

**实现文件：** `ui/main_window.py` - `DropArea` 类

**功能：**
- 拖拽文件/目录到窗口
- 自动识别并设置输入路径
- 悬停时高亮提示（虚线边框变色）
- 自动加载文件预览

**代码量：** ~100 行

#### 📋 文件列表预览

**实现文件：** `ui/file_preview_widget.py`

**功能：**
- 显示目录中的所有支持文件
- 实时文件计数
- 双击查看文件路径
- 刷新/清空操作
- 支持格式：PDF, PNG, JPG, JPEG, TIFF, BMP

**代码量：** ~140 行

**特性：**
- 独立可复用组件
- 文件类型过滤
- 工具提示显示完整路径
- 选中高亮

#### 🕐 处理历史记录

**实现文件：** `ui/history_manager.py`

**功能：**
- 自动记录每次处理
- JSON 格式持久化存储
- 保留最近 50 条记录
- 统计信息（成功率、总文件数）
- 查看历史记录对话框

**代码量：** ~90 行

**存储位置：** `data/history.json`

**记录内容：**
- 时间戳
- 输入/输出目录
- 处理选项
- 成功/失败状态
- 处理文件数

## 🚀 使用指南

### 快速开始

1. **运行应用：**
   ```bash
   python main.py
   ```

2. **选择目录：**
   - 点击"选择输入"按钮，或
   - 直接拖拽目录到拖拽区域

3. **配置选项：**
   - 勾选需要的处理功能
   - 选择处理方法

4. **开始处理：**
   - 点击"开始处理"按钮
   - 查看进度和日志

### 高级功能

**查看历史记录：**
- 点击"历史记录"按钮
- 查看最近 10 条处理记录

**刷新文件预览：**
- 点击文件列表右上角的"刷新"按钮

**清空文件列表：**
- 点击文件列表底部的"清空"按钮
