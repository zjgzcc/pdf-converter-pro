## 🧩 组件说明

### 1. 拖拽区域 (DropArea)

**功能：** 支持拖拽文件/目录快速导入

**特性：**
- 虚线边框，悬停时高亮
- 大图标 + 提示文字
- 自动识别文件/目录
- 加载文件预览

**样式：**
```css
QWidget#drop_area {
    background-color: #ffffff;
    border: 3px dashed #6c5ce7;
    border-radius: 15px;
    padding: 30px;
}
```

### 2. 文件预览 (FilePreviewWidget)

**功能：** 显示目录中的文件列表

**特性：**
- 实时文件计数
- 双击查看文件
- 刷新/清空操作
- 支持文件类型过滤

**支持格式：** `.pdf`, `.png`, `.jpg`, `.jpeg`, `.tiff`, `.bmp`

### 3. 历史记录 (HistoryManager)

**功能：** 记录处理历史

**特性：**
- 自动保存最近 50 条记录
- JSON 格式存储
- 统计信息（成功率、总文件数）
- 快速查看最近记录

**存储位置：** `data/history.json`

### 4. 进度条 (ProgressBar)

**样式：**
```css
QProgressBar::chunk {
    background: qlineargradient(x1:0, y1:0, x2:1, y2:0,
        stop:0 #6c5ce7, stop:1 #a29bfe);
}
```

**状态：**
- 0%: 未开始
- 10-30%: 去水印
- 40-60%: OCR 识别
- 70-90%: WORD 转换
- 100%: 完成

## 🎭 交互流程

### 标准流程

1. **选择输入目录** → 自动加载文件预览
2. **选择输出目录** → 准备处理
3. **配置选项** → 选择处理方式
4. **点击开始** → 显示进度
5. **查看日志** → 了解处理详情
6. **完成** → 记录历史

### 快捷流程（拖拽）

1. **拖拽目录/文件** → 自动设置输入
2. **选择输出目录** → 准备处理
3. **点击开始** → 处理

## 📱 响应式设计

### 最小窗口尺寸

- **宽度：** 1000px
- **高度：** 700px

### 布局策略

- 使用 `QSplitter` 实现可调节的上下区域
- 上部（拖拽 + 选项）：60% 高度
- 下部（预览 + 日志）：40% 高度

### 自适应

- 按钮使用最小宽度，随窗口扩展
- 列表和日志区域可伸缩
- 状态栏固定底部
