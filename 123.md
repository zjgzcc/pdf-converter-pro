## 🚀 5 分钟快速启动

### 1. 安装依赖

```bash
# 进入项目目录
cd pdf-converter-pro

# 安装 Python 依赖
pip install -r requirements.txt
```

### 2. 安装系统依赖

**Windows:**
```bash
# 运行一键安装脚本
scripts\install.bat
```

或手动安装：
- **Tesseract OCR**: https://github.com/UB-Mannheim/tesseract/wiki
  - 安装时勾选中文语言包
  - 添加到系统 PATH
- **Poppler** (PDF 转图片): https://github.com/oschwartz10612/poppler-windows/releases

### 3. 打开 GUI 界面

```bash
# 方法 1: 直接运行
python main.py

# 方法 2: 使用命令行工具
python cli.py process --gui
```

**GUI 功能:**
- 🧹 去水印（auto/opencv/iopaint）
- 🔍 OCR 识别（转可搜索 PDF）
- 📝 WORD 转换（保持格式布局）
- 支持拖拽文件
- 批量处理
- 实时进度显示

### 4. 命令行使用

```bash
# 处理单个 PDF
python cli.py process document.pdf

# 批量处理 + 去水印
python cli.py process ./input_pdfs -o ./output --watermark

# 仅 OCR
python cli.py process ./pdfs --ocr-only

# 仅转 WORD
python cli.py process ./pdfs --word-only
```

## 🔧 核心功能说明

### 1. 去水印模块 (`core/watermark.py`)

**支持 3 种模式:**
| 模式 | 速度 | 质量 | 适用场景 |
|------|------|------|----------|
| `auto` | 中等 | 智能选择 | 推荐，自动判断 |
| `opencv` | 快 (~50ms/张) | 良好 | 批量处理、简单水印 |
| `iopaint` | 慢 (~2s/张) | 优秀 | 复杂水印、高质量 |

**使用示例:**
```python
from core.watermark import WatermarkRemover

remover = WatermarkRemover(method="auto")
remover.remove(Path("input.png"), Path("output.png"))
```

**待优化:**

### 2. OCR 模块 (`core/ocr.py`)

**支持 3 种引擎:**
| 引擎 | 速度 | 中文准确率 | 适用场景 |
|------|------|------------|----------|
| `ocrmypdf` | 快 | 好 | 扫描 PDF 转可搜索 |
| `tesseract` | 中 | 一般 | 简单文档 |
| `paddleocr` | 中 | 优秀 | 复杂版面、表格 |

**使用示例:**
```python
from core.ocr import quick_ocr, batch_ocr

# 单文件
quick_ocr(Path("scan.pdf"))

# 批量（4 线程）
batch_ocr(Path("input/"), Path("output/"), max_workers=4)
```

**待优化:**
- [ ] 添加更多语言包支持
- [ ] 优化 PaddleOCR 模型加载
- [ ] 添加 GPU 加速选项

---

### 3. WORD 转换模块 (`core/converter.py`)

**支持 3 种方法（自动降级）:**
1. **PaddleOCR** - 版面恢复最好，支持表格/图片/多栏
2. **pdf2docx** - 轻量快速，适合简单文档
3. **FreeP2W** - 公式支持好

**使用示例:**
```python
from core.converter import PDF2WordConverter

converter = PDF2WordConverter(method="paddleocr")
converter.convert(Path("input.pdf"), Path("output.docx"))
```

## ❓ 常见问题

### Q: GUI 打不开，报错 `No module named 'PyQt6'`
**A**: 重新安装依赖
```bash
pip install -r requirements.txt --force-reinstall
```

### Q: OCR 识别中文乱码
**A**: 安装 Tesseract 中文语言包
- 下载 `chi_sim.traineddata`
- 放到 `C:\Program Files\Tesseract-OCR\tessdata\`

### Q: 转换速度慢
**A**: 
1. 减少 OCR 引擎数量（只用 1 种）
2. 降低 DPI 设置（config.json 中调整）
3. 使用 GPU 加速（需要 NVIDIA 显卡）

### Q: 去水印效果不好
**A**: 
1. 尝试切换模式（auto → opencv → iopaint）
2. 调整水印检测参数
3. 复杂水印建议用 IOPaint
