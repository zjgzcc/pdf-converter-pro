## 🚀 5 分钟快速启动

### 方式 1: 双击启动（最简单）

**Windows:** 双击项目目录中的 `start.bat` 文件

或者桌面快捷方式：**PDF Converter Pro.lnk**

### 方式 3: 一键安装脚本

```bash

```

## 🖥️ GUI 界面使用

**启动:**
```bash
python main.py
```

**功能:**
- 🧹 **去水印** - 支持 auto/opencv/iopaint 三种模式
- 🔍 **OCR 识别** - 转可搜索 PDF，3 种引擎可选
- 📝 **WORD 转换** - 保持原格式和布局
- 📥 **拖拽支持** - 直接拖入 PDF/图片
- ⚡ **批量处理** - 支持文件夹批量
- 📊 **实时进度** - 显示处理状态

**操作流程:**
1. 选择输入目录（或拖拽文件）
2. 选择输出目录
3. 勾选需要的功能（可组合）
4. 点击"开始处理"

## 💻 命令行使用

```bash
# 处理单个 PDF
python cli.py process document.pdf

# 批量处理 + 去水印
python cli.py process ./input_pdfs -o ./output --watermark

# 仅 OCR
python cli.py process ./pdfs --ocr-only

# 仅转 WORD
python cli.py process ./pdfs --word-only

# 查看帮助
python cli.py --help
```

## 🔧 核心功能说明

### 1. 去水印模块 (`core/watermark.py`)

**3 种模式:**
| 模式 | 速度 | 质量 | 适用 |
|------|------|------|------|
| `auto` | 中等 | 智能选择 | 推荐 |
| `opencv` | 快 | 良好 | 简单水印 |
| `iopaint` | 慢 | 优秀 | 复杂水印 |

### 2. OCR 模块 (`core/ocr.py`)

**3 种引擎:**
| 引擎 | 速度 | 中文准确率 |
|------|------|------------|
| `ocrmypdf` | 快 | 好 |
| `tesseract` | 中 | 一般 |
| `paddleocr` | 中 | 优秀 |

### 3. WORD 转换 (`core/converter.py`)

**自动降级链:** PaddleOCR → pdf2docx → FreeP2W

## ❓ 常见问题

### Q: GUI 打不开，报错 `No module named 'PyQt6'`
**A**: 
```bash
pip install -r requirements.txt
```

### Q: OCR 中文乱码
**A**: 安装 Tesseract 中文语言包
- 下载：https://github.com/UB-Mannheim/tesseract/wiki
- 放到 `C:\Program Files\Tesseract-OCR\tessdata\`

### Q: 转换速度慢
**A**: 
1. 只用 1 种 OCR 引擎
2. 降低 DPI（config.json）
3. GPU 加速（需要 NVIDIA 显卡）
